# External Specification Data Notices

## Webref CSS

The machine-readable Webref CSS package is produced by the [W3C Webref project](https://github.com/w3c/webref) and distributed under the MIT License. The exact package version and repository commit are recorded in `spec-sources.json`.

Registry rows derived from WHATWG specifications retain links to their source definitions and are attributed under CC BY 4.0. Omena transforms those definitions into machine-readable audit records and does not republish a modified specification. Entries originating in W3C specifications are used as reference data under the W3C Document License.

## MDN Browser Compatibility Data

Compatibility fallback data comes from [MDN Browser Compatibility Data](https://github.com/mdn/browser-compat-data), released under CC0 1.0 Universal. The pinned package is recorded in `spec-sources.json`. It is compatibility evidence rather than specification authority.
